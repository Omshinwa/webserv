CGI Upload — Uppercase

POST a text file's contents → CGI uppercases it → saves to uploads/

← Home

The request body is the file content. The save name is sent in the query string, which the server hands to the script as QUERY_STRING: POST /cgi-bin/uppercase.py?name=<file>
